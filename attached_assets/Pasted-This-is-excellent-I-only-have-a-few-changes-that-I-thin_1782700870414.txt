This is excellent. I only have a few changes that I think will make it even more future-proof.

1. Don't store scan status on every file

This is the one thing I would change.

Instead of storing:

🟢 Unchanged
🟡 Modified
🔵 New
🟣 Moved
🔴 Deleted

forever...

I'd make those Last Scan Results, not permanent states.

Why?

Because after the next scan, a "New" file isn't new anymore.

Instead I'd store something like:

lastScanAction

NEW
MODIFIED
MOVED
UNCHANGED
DELETED
VERIFIED

Then also keep:

lastScannedAt

2026-06-28 14:18

That gives you historical context without misleading users months later.
