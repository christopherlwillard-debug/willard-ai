One architectural recommendation

The only thing I'd add isn't a new feature—it's a cross-cutting design principle.

Create a "single source of truth"

Everything should build on the same library record.

For every media item, imagine one canonical record:

Media Item

Identity

Path(s)

Metadata

AI Metadata

OCR

Embeddings

Collections

Timeline

GPS

People

Notes

Favorites

Duplicates

Thumbnail

Preview

Then:

The Scanner (#79) populates it.
The Watcher (#82) keeps it up to date.
Search (#81) queries it.
Media Details (#83) displays it.

No feature should maintain its own separate copy of metadata.

That architectural rule will save you an enormous amount of complexity as the project grows.
