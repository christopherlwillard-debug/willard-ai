#49 - Library Health & Cleanup

What & Why

Over time, media libraries naturally accumulate stale records, orphaned thumbnails, empty folders, and missing files as users organize their NAS outside of the Media Center.

The Library Health system keeps the Media Center synchronized with the NAS and provides one-click maintenance tools to repair common issues.

The NAS remains the source of truth.

No original media files are modified or deleted without explicit user approval.

One change I'd make

I actually wouldn't put this button directly on the Library toolbar.

The Library toolbar should stay focused on browsing:

Grid
Timeline
Map
Filters
Search

Maintenance belongs in a dedicated Library Health page (or under Settings → Library Health). The dashboard can simply show a small health indicator:

🟢 Library Healthy
🟡 12 Issues Found

Clicking that takes you to the full Health Center where users can review issues and choose which maintenance tasks to run.

That keeps the browsing experience clean while still making maintenance easy to find when it's needed.
